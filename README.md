# TRON 私钥碰撞工具

一个基于Python的TRON区块链私钥碰撞检测工具，通过随机生成私钥并检查对应地址的资产余额来寻找有价值的钱包。

## 🚀 功能特性

- **随机私钥生成**: 使用安全的随机数生成32字节私钥
- **TRON地址计算**: 从私钥生成对应的TRON Base58Check格式地址
- **多代币支持**: 检测TRX、TRC10和TRC20代币（包括USDT等）
- **异步查询**: 使用异步HTTP请求提高查询效率
- **实时日志**: 实时显示查询进度和结果
- **自动保存**: 自动保存发现的有资产地址和对应私钥
- **速率限制**: 遵守API限制，每秒最多10次查询

## 🛠️ 技术原理

1. **私钥生成**: 使用`os.urandom(32)`生成随机32字节私钥
2. **公钥推导**: 通过椭圆曲线算法（secp256k1）从私钥生成公钥
3. **地址计算**: 对公钥进行Keccak-256哈希，添加TRON前缀0x41，最后进行Base58Check编码
4. **余额查询**: 调用TronGrid公开API查询地址余额和代币信息

## 🛠️ 故障排除

### 常见问题

#### 403错误处理
如果程序运行时遇到403报错，这通常是由于API访问限制导致的。解决方法：

1. **更换IP地址**
   - 使用VPN切换到其他地区
   - 重启路由器获取新的IP地址
   - 使用移动热点

2. **等待一段时间**
   - TronGrid API有速率限制
   - 建议等待几分钟后再次尝试

3. **检查网络连接**
   - 确保网络连接稳定
   - 尝试访问 https://api.trongrid.io 确认API可访问

#### 其他问题

- **程序无响应**: 检查网络连接，确保能访问TronGrid API
- **日志文件未生成**: 确保程序有写入权限，至少运行一次程序
- **找不到non_zero_addresses.txt**: 这是正常的，该文件只有在发现有资产地址时才会生成
